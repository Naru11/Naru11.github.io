2. 核心语法

2.1 变量:

- `var`用来声明变量的关键字
- ```let```用来声明变量的关键字
- ```const```用来定义常量(不可更改)

>不声明变量直接使用，容易报错

2.2 变量类型:

  • number:数值变量
  • string：字符串
  • boolean：布尔值
  • object：对象，类似指针，e.g.{1,2,3}
  • undefined：未定义

2.3 运算符:

  • 等于: ===
  • 不等于: !==
  • typeof：返回变量类型。

2.4 String对象

  • 字符串长度：str.length
  • 字符串拼接：str1+str2
  • 字符串截取：str.substring(start,end)
  • 字符串转数值：parseInt(str)
  • charAt(index)：返回字符串中指定位置的字符
  • indexOf(str,index)：返回字符串中指定字符串第一次出现的位置
  • split：将字符串分割为字符串数组

2.5 数组

  • 创建数组： var arr = new Array(size);
  • 为数组赋值：var arr = new Array(1,2,3);
  • 访问数组：arr[index];

2.6 输入输出

  • 输入
    获取标签内容（括号内可以是标签或类名）：
    int input=document.querySelector("textarea")
    内容包含在：input.value里。

  • 输出
    用output.innerHTML=input.value来把输入的内容输出。

  • 字符型转数值型: a=parseInt(a)

  • 保留两位小数：x.tofixed(2)

格式化字符串

console.log(内容)
查看输出的内容：在html上按F12查看。

2.7 对象

  • 调用方式：person['name']orperson.name。
  • 给对象增加元素： person.school='sztu'orperson['school']='sztu'

2.8 类

  • 继承
    • super：指向父类对象
    • this：指向本身实例
    • 父类跟子类拥有相同函数名时，子类会把父类覆盖掉。
  • 静态函数：只能通过类名来访问
  • 静态变量
    Point.cnt=0所有类的实例共用，写在类外

4. 事件

通过addEventListener函数作为元素绑定事件的触发函数

  • 第一个参数是事件，第二个是触发的函数。 通过document.querySelector把css和js绑定起来，可以给css加入函数 常见的触发函数
  • 鼠标
    • click：单击
    • dbclick：双击
    • contextmenu：右键点击
    • mousedown：鼠标按下
      • event.button :0，1，2分别表示左键中键右键
    • mouseup ：鼠标弹起
  • 键盘
    • keydown ：某个键是否被按住，事件会连续触发
      • event.code :返回按那个键
      • event.altkey、event.ctrlkey、event.shiftkey分别表示是否按下对应的键，返回true false。
    • keyup:键盘弹起
    • keypress
  • 表单
    • focus:聚焦某个元素，比如鼠标点击输入框
    • blur：取消聚焦，比如鼠标点击输入框后移开
    • change：内容发生改变，比如修改输入框的内容
  • 窗口
    • resize:窗口大小发生变化
    • scroll:滚动
    • load:加载完成

5. 常用库

  • JQuery
    在元素中添加：
    • $('div')选择前端元素
    • $div.on("click",function(){})绑定函数
    • $div.off("click",function(){})取消绑定函数
    • return false：阻止子事件和父事件。
    • .hide() .show()：隐藏展现。 - 创建元素let $a =$('<a></a>')
      • 将创建的元素加入原有的$div.append($a)
      • 清空$div.empty()
      • 删除单个$div.remove($a)
      • 更改css样式$div.css('background-color','orange');
      • 对类、css、html等操作
  • ajax
    • get方法
    • post方法
  • setTimeout setInterval
  • requestAnimationFrame
  • MAP
    • let map = new Map()
    • map.set('name','xm')
  • Set
    • let set = new Set()
    • set.add('xm')
  • Storage
  • JSON
