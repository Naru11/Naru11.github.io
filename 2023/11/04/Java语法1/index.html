Java语法

发表于2023-11-04|更新于2023-11-04|语法
|阅读量:

1. 变量、运算符、输入与输出

内置数据类型:

byte short int long float double boolen char

常量：使用final修饰。

  • final int N = 110

类型转换：

  • 显式转换：int x = (int)'A';
  • 隐式转换：double x = 12, y = 4 * 3.3;

输入

  • 方式一:效率较低，适合输入规模小使用
  • 方式二:效率较高，输入规模较大时使用。注意需要抛异常。

输出

  • 方式1：效率较低，输出规模较小时使用。

    println会在最后加换行符，print则不会。

  • 方式2，效率较高，输出规模较大时使用。注意需要抛异常。

与C++类似

if-else表达式 switch语句 逻辑表达式和条件表达式 循环语句

数组

初始化

  • 初始化长度为5的int数组，初始值为0：int[] a = new int[5]
  • 初始化长度为3的char数组，初始值为：'a', 'b', 'c' char[] c = {'a', 'b', 'c'};
  • d与c地址相同，更改c中的元素，d中的元素也会改变: char[] d = c;
  • 多维数组：int[][] a = new int[2][3];

常用函数：

  • 属性length：返回数组长度，注意不加小括号
  • Arrays.sort()：数组排序
  • Arrays.fill(int[] a, int val)：填充数组
  • Arrays.toString()：将数组转化为字符串
  • Arrays.deepToString()：将多维数组转化为字符串
    数组不可变长

字符串

初始化：

string转double：double money = Double.parseDouble(money_str)
int转String：String.format("(%d, %d)", x, y);

  • 读取String中的字符：

常用函数

  • length()：返回长度
  • split(String regex)：分割字符串
  • indexOf(char c)、indexOf(String str)：查找，找不到返回-1
  • equals()：判断两个字符串是否相等，注意不能直接用==
  • compareTo()：判断两个字符串的字典序大小，负数表示小于，0表示相等，正数表示大于
  • startsWith()：判断是否以某个前缀开头
  • endsWith()：判断是否以某个后缀结尾
  • trim()：去掉首尾的空白字符
  • toLowerCase()：全部用小写字符
  • toUpperCase()：全部用大写字符
  • replace(char oldChar, char newChar)：替换字符
  • replace(String oldRegex, String newRegex)：替换字符串
  • substring(int beginIndex, int endIndex)：返回(beginIndex, endIndex)中的子串

函数

Java的所有变量和函数都要定义在类中。

函数或变量前加static表示静态对象，类似于全局变量。
静态对象属于class，而不属于class的具体实例。

静态函数中只能调用静态函数和静态变量。

类

源文件声明规则

  • 一个源文件中只能有一个public类。
  • 一个源文件可以有多个非public类。
  • 源文件的名称应该和public类的类名保持一致。
  • 每个源文件中，先写package语句，再写import语句，最后定义类。

类的继承

每个类只能继承一个类。

接口

interface与class类似。主要用来定义类中所需包含的函数。

接口也可以继承其他接口，一个类可以实现多个接口。

定义

接口的继承：interface Hero extends Role
接口的实现：一个类可以实现多个接口
class Zeus implements Hero

常用容器

  • List:java.util.List<>
  • 栈：java.util.Stack<>
  • 队列：java.util.Queue<>
  • set:java.util.Set<K>
  • map:java.util.Map<K, V>
